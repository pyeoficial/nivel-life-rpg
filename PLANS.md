# Plan ejecutable del MVP NIVEL

## Objetivo

Entregar una PWA instalable y demostrable que convierta acciones reales en misiones, XP, monedas, evolución de personaje y recompensas conscientes.

## Fase 1 — Núcleo visible

- [x] Dirección visual fantasy mobile-game original.
- [x] Misiones diarias con XP y monedas.
- [x] Héroe con cuatro etapas evolutivas.
- [x] Tienda de Caprichos y progreso semanal.
- [x] Planificador según energía y tiempo.

## Fase 2 — Producto real

- [x] Persistencia D1 por sesión anónima.
- [x] Reglas económicas calculadas en servidor.
- [x] Historial de eventos y control de doble recompensa.
- [x] Manifest, iconos, service worker e instalación PWA.

## Fase 3 — Validación y entrega

- [ ] Generar y aplicar migración.
- [ ] Lint y build de producción.
- [ ] Probar: misión → nivel → tienda → canje → recarga.
- [ ] Validar vista móvil e instalación.
- [ ] Publicar checkpoint funcional.

## Después del hackathon

1. Cuenta y sincronización multidispositivo.
2. Creación libre de hábitos, misiones y recompensas.
3. Recordatorios inteligentes y calendario.
4. Planificador opcional con IA y límites de costo.
5. Más avatares, cosméticos y temporadas.
6. Métricas de retención, bienestar y control de sobrejustificación.

## Iteración 2 — NIVEL como juego diario

- [x] Sustituir el tablero convencional por un Mapa de Aventura.
- [x] Añadir Rituales con seguimiento de siete días y registro en un toque.
- [x] Incorporar hábitos iniciales: ejercicio, sueño, puntualidad y lectura.
- [x] Permitir crear misiones y rituales desde un único botón central.
- [x] Añadir check-in de energía y ánimo.
- [x] Añadir Cofre del Día con validación del servidor.
- [x] Reorganizar recompensas y progreso dentro del Campamento.
- [x] Reiniciar misiones diariamente sin destruir el historial general.
- [x] Migrar automáticamente partidas de la versión inicial.
